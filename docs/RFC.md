# System design

Сервис для PayMeMore — оценивание зарплаты по резюме

- Пользователь
- Резюме

## 1. Требования

- Обязательные функциональные требования
  - Пользователь может рассчитать вилку ЗП по:
    - Роли
    - Опыту
    - Навыкам
    - Региону
  - Пользователь может получить по рекомендации по улучшению резюме от LLM
  - Пользователь можем перерассчитать вилку ЗП с другими параметрами 
  
- Дополнительные функциональные требования
  - Система сохраняет результаты поиска вилки ЗП и рекомендаций, чтобы пользователь мог просмотреть это
  - Пользователь может импортировать резюме из PDF
  - При заполнении резюме система предлагает актуальные скиллы для введенной роли
  - Система отправляет метрики удовлетворенности расчетом вилки ЗП на бэкенд
  - Администратор платформы может управлять пользователями с зарегистрированными пользователями
  - Пользователь может переключать темную/светлую тему

- Нефункциональные требования
	- MVP
		- DAU: 300 пользователей
		- RPS 
    - SLA: 90% (Возможный downtime 36 дней в год)
